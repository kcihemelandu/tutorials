# Shortcuts

- ctrl + P (Command Pallete)
  - Find a file
  - Typing the right angle bracket
  - using the at symbol @
  - using in hash #
  - :line_number
- ctrl+f
- ctrl+d
- Using alt+click key
- Alt + arrow to move
- Alt + Shift + arrow -> Copy
- CTRL + L Highlight line

- CTRL + ` to open terminal
- CTRL + K to clear terminal

- Preferences: Configure User Snippets
